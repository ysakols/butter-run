Butter Run — Complete Screen Audit & Flows

Every screen, every button, every dialog, every flow. Nothing missing.

1. App Launch Flow

App Launch
  │
  ├─ Database Error? ──── YES ──→ [Error Screen] "Something went wrong"
  │                                   └─ (Dead end, user must reinstall)
  │
  └─ NO ──→ Check UserProfile exists?
              │
              ├─ NO ──→ [Onboarding Walkthrough]
              │            Page 1: Welcome
              │            Page 2: Butter Zero
              │            Page 3: Churn Tracker
              │            Page 4: Profile Setup
              │              └─ "Let's Churn" ──→ Create Profile ──→ [Main Tab View]
              │
              └─ YES ──→ Check for Crash Draft?
                           │
                           ├─ Draft Found ──→ [Alert: "Unfinished Run"]
                           │                    ├─ "Discard" ──→ Delete Draft ──→ [Main Tab View]
                           │                    └─ (dismiss) ──→ [Main Tab View]
                           │
                           └─ No Draft ──→ [Main Tab View]

2. Main Navigation

[Main Tab View]
  │
  ├─ Tab 1: Run ──────→ [Home Screen]
  ├─ Tab 2: History ──→ [Run History]
  ├─ Tab 3: Guide ────→ [Churn Guide]   ← NEW tab (was not in original app)
  └─ Tab 4: Settings ─→ [Settings]

3. Start Run Flow

[Home Screen]
  │
  ├─ Toggle "Butter Zero" ON/OFF
  ├─ Toggle "Churn Tracker" ON/OFF
  │
  └─ Tap "Start Churning"
       │
       ├─ Churn OFF ──→ [Active Run] (direct start)
       │
       └─ Churn ON ──→ [Churn Setup Sheet]
                         │  - Select cream type (Heavy/Whipping)
                         │  - Enter amount (cups)
                         │  - Toggle room temperature (shows warning alert)
                         │
                         ├─ "Start Churning" ──→ [Active Run] with churn config
                         └─ "Cancel" ──→ back to [Home]

4. Active Run Flow

[Active Run]
  │
  ├─ GPS Banners (automatic)
  │    ├─ Weak signal: orange banner
  │    └─ Lost signal: red banner
  │
  ├─ Auto-Pause Banner (automatic when stopped moving)
  │
  ├─ Map/Metrics Toggle ──→ switch between map view and metrics grid
  │
  ├─ 🧈 EAT button (only if Butter Zero ON)
  │    └─ [Eat Butter Sheet]
  │         ├─ "1 tsp" (34 cal) ──→ add + dismiss + show undo toast
  │         ├─ "1 pat" (36 cal) ──→ add + dismiss + show undo toast
  │         ├─ "1 tbsp" (102 cal) ──→ add + dismiss + show undo toast
  │         ├─ "½ stick" (408 cal) ──→ add + dismiss + show undo toast
  │         ├─ Custom tsp + "Add" ──→ add + dismiss + show undo toast
  │         └─ "Cancel" ──→ dismiss
  │
  ├─ Undo Toast (appears 8 sec after eating)
  │    └─ "Undo" ──→ removes last butter entry
  │
  ├─ ⏸ PAUSE/RESUME button ──→ toggles run state
  │
  └─ ⏹ STOP button (hold 3 sec)
       └─ [Confirmation: "End run?"]
            ├─ "Finish Run" ──→ save run ──→ [Run Summary]
            └─ "Cancel" ──→ back to [Active Run]

5. Post-Run Flow

[Run Summary]
  │
  ├─ View: Butter animation, stats, splits, Butter Zero score, churn result
  │
  ├─ "Share My Run" ──→ select format (Story 9:16 / Square 1:1)
  │                       └─ [iOS Share Sheet] ──→ share to apps
  │
  └─ "Done" ──→ return to [Home Screen]

6. History Flow

[Run History]
  │
  ├─ Empty state (no runs) ──→ "No runs yet. Tap Run to start!"
  │
  ├─ Tap any run row ──→ [Run Detail View]
  │                        │
  │                        ├─ View: Butter hero, map, stats, splits, butter entries
  │                        │
  │                        └─ "Edit" (toolbar) ──→ [Run Edit View]
  │                                                 │  - Edit distance
  │                                                 │  - Edit notes
  │                                                 │
  │                                                 ├─ "Save Changes" ──→ update + dismiss
  │                                                 └─ "Cancel" ──→ dismiss
  │
  ├─ Swipe-to-delete any run ──→ [Confirmation: "Delete this run?"]
  │                                ├─ "Delete" ──→ remove run
  │                                └─ "Cancel" ──→ dismiss
  │
  └─ "+ Manual" button ──→ [Manual Run Entry]
                            │  - Date picker
                            │  - Distance input
                            │  - Duration (minutes)
                            │  - Unit picker (mi/km)
                            │  - Shows estimated pats preview
                            │
                            ├─ "Save Run" ──→ create manual run ──→ dismiss
                            └─ "Cancel" ──→ dismiss

  │    ├─ Version ──→ display only

All Modals, Sheets & Confirmation Dialogs

Every overlay, alert, and sheet in the app with exact behavior.

⚠️ Delete All Data — Confirmation

Delete All Data?

This will permanently delete all your runs, achievements, and profile. This cannot be undone.

Delete Everything
Cancel

After "Delete Everything":
All Runs, Achievements, RunDrafts, and UserProfile are deleted. App restarts and shows Onboarding page 1 (profile no longer exists).
🗑️ Delete Run — Confirmation

Delete this run?

This run and all its data will be permanently removed.

Delete
Cancel

After "Delete":
Run removed from SwiftData. User stays on History screen. Run disappears from list.
🏁 End Run — Confirmation

End run?

Your run will be saved.

Finish Run
Cancel

After "Finish Run":
Run saved to SwiftData. Draft deleted. Full-screen Run Summary appears.
💾 Crash Recovery — Alert

Unfinished Run

Found an unfinished run from [date]. Duration: [time].

Discard

When shown:
App crashed or was killed during a run. Draft was saved. On next launch, user can discard it.
⚖️ Recalculate — Alert

Recalculate Past Runs?

Your weight changed. Want to update calorie estimates for all past runs?

Recalculate
Skip

After "Recalculate":
All past runs' butter/calorie values are recalculated with new weight. User stays on Settings.
🌡️ Room Temp Warning — Alert

Room Temperature

Room temperature cream won't churn properly. For best results, use cold cream. Progress will be capped at Whipped.

OK

When shown:
User toggles "room temperature" in Churn Setup. Warning only — doesn't block.

Sheets (Half-Screen Modals)

🧈 Eat Butter Sheet
Trigger: Tap Eat button during active run
Presentation: .medium detent (half screen)
Contents: 4 preset buttons (tsp, pat, tbsp, ½ stick) + custom input
On select: Adds butter entry, dismisses, shows undo toast (8 sec)
Cancel: Toolbar X button dismisses
🫧 Churn Setup Sheet
Trigger: "Start Churning" when Churn toggle is ON
Presentation: .medium detent
Contents: Cream type picker, amount, room temp toggle
On start: Creates config, dismisses, opens Active Run after 0.3s
Cancel: Toolbar X button dismisses
✍️ Manual Run Entry
Trigger: "+ Manual" button in History
Presentation: Standard sheet
Contents: Date, distance, duration, unit picker
Shows: Estimated pats preview as you type
On save: Creates run with isManualEntry flag, dismisses
Disabled: Save disabled if distance=0 or duration=0
✏️ Run Edit View
Trigger: "Edit" button on Run Detail
Presentation: Standard sheet
Contents: Distance (editable), notes (editable). Date/duration read-only.
On save: Updates run, recalculates pace, dismisses
Cancel: Toolbar X button discards changes

Full-Screen Covers

🏃 Active Run (Full Screen)
Trigger: Start run from Home
Presentation: .fullScreenCover (no swipe dismiss)
Exit: Only via Stop → Confirm → Finish
On dismiss: Clears churn config
📊 Run Summary (Full Screen)
Trigger: After finishing a run
Presentation: .fullScreenCover
Exit: "Done" toolbar button
Returns to: Home Screen
